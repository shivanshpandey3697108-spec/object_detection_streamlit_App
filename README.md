# YOLOv8 Streamlit Object Detection App

This project is a real-time object detection application built using YOLOv8 and Streamlit.

## Features
- Image object detection
- Video object detection
- Webcam live detection
- Fast inference using YOLOv8 Nano

## Installation

Clone the repository:

git clone https://github.com/shivanshpandey3697108-spec/object_detection_streamlit_App.git

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py

## Technologies Used
- Python
- YOLOv8 (Ultralytics)
- Streamlit
- OpenCV
